PROJECT OVERVIEW
================================================================================
This is a full-stack web application with:
- Frontend: React with Vite
- Backend: Django REST Framework
- Database: PostgreSQL
- Containerization: Docker & Docker Compose

================================================================================
SYSTEM REQUIREMENTS
================================================================================

OPTION 1: Using Docker (RECOMMENDED - Easiest)
---------------------------------------------
- Docker Desktop (v4.0 or higher)
- Docker Compose (included with Docker Desktop)
- 4GB RAM minimum
- 2GB free disk space

OPTION 2: Manual Setup (Without Docker)
---------------------------------------
- Python 3.8 or higher
- Node.js 16 or higher (with npm)
- PostgreSQL 15 or higher
- 4GB RAM minimum
- 2GB free disk space

================================================================================
QUICK START - OPTION 1: DOCKER (RECOMMENDED)
================================================================================

1. Ensure Docker Desktop is running

2. Navigate to project root directory:
   cd starevents-management

3. Start all services:
   docker-compose up

4. Services will be available at:
   - Backend API: http://localhost:8000
   - Frontend: http://localhost:3000
   - Database: localhost:5432

5. To stop services:
   docker-compose down

================================================================================
QUICK START - OPTION 2: MANUAL SETUP
================================================================================

BACKEND SETUP:
--------------
1. Install Python 3.8+ from https://www.python.org/

2. Create virtual environment:
   python -m venv venv
   
3. Activate virtual environment:
   Windows: venv\Scripts\activate
   macOS/Linux: source venv/bin/activate

4. Install dependencies:
   cd backend
   pip install -r requirements.txt

5. Create .env file in backend directory:
   DEBUG=True
   DATABASE_URL=postgresql://starevents_user:starevents_pass@localhost:5432/starevents_db

6. Run migrations:
   python manage.py migrate

7. Create superuser:
   python manage.py createsuperuser

8. Start backend server:
   python manage.py runserver
   Backend runs at: http://localhost:8000

FRONTEND SETUP:
---------------
1. Install Node.js 16+ from https://nodejs.org/

2. Navigate to frontend directory:
   cd frontend

3. Install dependencies:
   npm install

4. Start development server:
   npm run dev
   Frontend runs at: http://localhost:3000

DATABASE SETUP (Manual Only):
-----------------------------
1. Install PostgreSQL 15 from https://www.postgresql.org/

2. Create database and user:
   CREATE DATABASE starevents_db;
   CREATE USER starevents_user WITH PASSWORD 'starevents_pass';
   ALTER ROLE starevents_user SET client_encoding TO 'utf8';
   ALTER ROLE starevents_user SET default_transaction_isolation TO 'read committed';
   ALTER ROLE starevents_user SET default_transaction_deferrable TO on;
   ALTER ROLE starevents_user SET default_transaction_level TO 'committed read';
   GRANT ALL PRIVILEGES ON DATABASE starevents_db TO starevents_user;

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Port 8000 or 3000 already in use
Solution: 
  - Kill the process using the port
  - Or modify port in backend (settings.py) or frontend (vite.config.js)

Issue: PostgreSQL connection failed
Solution:
  - Check PostgreSQL is running
  - Verify credentials in .env file
  - Ensure DATABASE_URL is correct

Issue: npm install fails
Solution:
  - Delete node_modules folder and package-lock.json
  - Clear npm cache: npm cache clean --force
  - Try again: npm install

Issue: Python dependencies fail to install
Solution:
  - Update pip: python -m pip install --upgrade pip
  - Try installing specific versions
  - Check Python version compatibility
